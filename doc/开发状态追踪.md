## 快速验证流程

```bash
# 1. 设置环境
cd /home/stellar/Projects/hardware_test_platform
./setup_env.sh
source venv/bin/activate

# 2. 列出可用 fixtures
./bin/run_fixture --list

# 3. 运行功能快速验证
./bin/run_fixture --name 功能快速验证

# 4. 运行单个 case
./bin/run_case --config cases/eth_case.json

# 5. 查看 CLI Dashboard（新终端）
python3 -m framework.dashboard
```

## 关键技术决策

1. **Python 3.8+**: 使用现代 Python 特性（dataclass、f-string 等）
2. **requirements.txt**: 简单直接的依赖管理
3. **虚拟环境**: 使用 venv 隔离项目依赖
4. **rich 库**: 用于 CLI Dashboard 的漂亮输出
5. **JSON 配置**: 配置驱动设计，无需修改代码
6. **状态码**: 统一错误处理，便于 TE/EE 理解
